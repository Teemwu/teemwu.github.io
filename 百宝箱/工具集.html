工具集合 #

收录一些常用且实用的工具。

常用工具 #

  1. Animate.css - CSS3 动画库
  2. Quicktype - 将 JSON 转换成各种语言模型或接口
  3. GlassUI - 生成 毛玻璃 效果
  4. Whirl - CSS3 Loading 效果库
  5. CSS Grid Generator - Grid 布局生成器
  6. Regular Expressions 101 - 正则测试
  7. CanIUse - 兼容性查询
  8. NpmTrends - npm 包比较
  9. Postman - 接口调试神器
  10. spy-debugger - web 无线真机调试神器

常用 NPM 包 #

  1. fs-extra - 文件增删改工具库，基于 fs 进行优化扩展
  2. commander - 命令行传参定义及解析工具库
  3. dayjs - 体积极小的日期时间处理库
  4. chalk - 终端输出字符串样式自定义
  5. axios - 基于 Promise 的网络请求库
  6. await-to-js - 异步等待包装器，以便于错误处理
  7. ora - 优雅的终端调试库，提供 loading 加载动画和个性化图标
  8. yalc - 本地开发组件库时 link 调试用
  9. leaflet - 一款地图交互库

Chrome 扩展 #

  1. Image downloader - Imageye - 图片下载神器
  2. uBlacklist - 屏蔽一些网站的搜索结果
  3. Absolute Enable Right Click & Copy - 强制开启一些页面的右击和复制功能

VSCode 扩展 #

  • Todo Tree - 自动查找项目标注了 TODO 字样的位置，并形成结构树方便查找，可在项目根目录增加 TODO.md 文件进行整体管理
  • Package Json Upgrade - 项目包更新提示，在 package.json 中显示可更新的版本号，按 cmd + . 即可选择更新
  • Paste Image - Markdown 文件中粘贴图片自动保存到本地指定目录，并自动生成图片的相对路径
  • Markdown All in One - 提高 markdown 编写效率工具，常用：选择文字粘贴网址自动补全、表格自动格式化、列表便捷操作

常用 APP #

  • vysor - 非常好用的一款安卓远程调试工具
  • DeepL - 翻译神器，翻译得比较接地气
  • GIF Brewery by Gfycat - MacOS 端比较好用的录频生成 gif 文件工具
  • snipaste - 截图神器

科学 #

  1. 猫云 - 便宜（每日提供免费节点 https://maoy.cc/mfdy)
  2. OKZ2 - 套餐多种
  3. FASTLINK - 价格较贵，质量较好（不过最近不大稳定）
