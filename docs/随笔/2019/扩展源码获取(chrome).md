# 扩展源码获取(chrome)

平时在使用别人的 chrome 扩展时总是觉得有些地方觉得做得不好，不过总体来说这个扩展对自己来说又必不可少，所以萌生了想改这个扩展的念头。

参考网友经验，可以通过以下几个步骤获取扩展源码：
- "chrome://version/" ->个人资料路径 (获取已安装扩展的目录路径)

- “chrome://extensions/”-> ID:XXXXXXXXXX(找到插件的ID)

- 根据目录和 ID 就可以找到扩展源码了
