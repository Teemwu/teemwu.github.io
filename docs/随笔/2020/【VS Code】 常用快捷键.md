# 【VS Code】 常用快捷键

#### 光标移动
| Mac | Windows | 说明 |
| :----: | :----------: | :-----: |
| Cmd + ← | Home | 光标移动到当前行最左侧 |
| Cmd + → | End    | 光标移动到当前行最右侧 |
| Cmd + ↑   | Ctrl + Home | 光标移动到文章的第一行 |
| Cmd + ↓   | Ctrl +End     | 光标移动到文章的最后一行 |
